# Simple server - client demo

### Goal

- Client must connect to server
- Client must emit an event
- Server must receive this, reply back with another event
- Client must receive this

### Getting started

- Server side
  - `Flask`
  - `flask-socketio`
- Client side
  - `python-socketio-client`
  - `websocket-client`

### Steps

- Run server
- Run client
